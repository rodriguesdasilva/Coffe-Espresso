Tecnologias Utilizadas
HTML5: Estrutura semântica do site.

CSS3: Estilização, Grid Layout, Flexbox e variáveis.

Google Fonts: Fontes Poppins e Roboto.

Icons8: Ícones para navegação e redes sociais.

Funcionalidades
Navegação Fixa: Menu superior que acompanha a rolagem da página.

Seção Hero: Destaque com imagem de fundo e chamada para ação (CTA).

Menu Dinâmico: Vitrine de produtos com efeito de hover interativo.

Avaliações: Seção de depoimentos de clientes com sistema de estrelas.

Mapa Integrado: Localização da loja via Google Maps.

Responsividade: Design adaptável para diferentes tamanhos de tela.

Como Rodar o Projeto
Clone o repositório:

Bash
git clone https://github.com/rodriguesdasilva/Coffe-Espresso.git
Acesse a pasta do projeto:

Bash
cd Coffe-Espresso
Abra o arquivo index.html no seu navegador.

Estrutura de Cores (CSS Variables)
O projeto utiliza variáveis CSS para manter a consistência visual:

--main-color: #d3ad7f (Dourado Café)

--bg: #010103 (Preto Profundo)

--black: #13131a (Preto Suave)
